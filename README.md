# OpusVox

**OpusVox** is a PWA communication assistant for people who have lost the ability to speak (ALS, stroke, tracheotomy, etc.). It transcribes a visitor's voice, generates contextual AI response suggestions, and speaks them aloud — optionally with the user's own cloned voice.

---

## Features

- 🎙️ **Voice transcription** — via Mistral Speech-to-Text API, or simulation mode without an API key
- 🤖 **Contextual response suggestions** — generated by Mistral AI with full conversation history
- 🔊 **Text-to-speech** — via Mistral TTS with voice cloning, or native Web Speech API fallback
- 🗣️ **Voice cloning** — record a 10-second sample or import an audio file to create a personalized voice profile
- 🌍 **Multilingual** — UI and TTS in French and English, with automatic visitor language detection
- ⌨️ **Keyboard shortcuts** — configurable key to start/stop recording (Space by default)
- 🔇 **Auto-stop** — recording stops automatically after 3 seconds of silence
- 📜 **History** — all conversations are saved locally and can be replayed
- 📱 **Installable PWA** — add to home screen on any device via browser install prompt

## Quick start

### Prerequisites

- Node.js 18+
- Yarn

### Development

```bash
yarn install
yarn dev
```

### Production (local)

```bash
yarn build
yarn preview
```

### Docker

```bash
docker-compose up -d
```

The app is served on `http://localhost:80` via nginx.

---

## Configuration

On first launch, an onboarding wizard guides the user through:

1. Choosing the interface language (FR / EN)
2. Entering a name and profile information
3. Providing a Mistral API key (optional — the app runs in simulation mode without one)

A Mistral API key is required for:
- Real voice transcription (STT)
- AI response generation
- TTS with voice cloning
- Voice cloning (paid Mistral plan required)

> **Privacy:** all data stays on your device. Only API calls to Mistral leave your browser.
